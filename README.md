# link_prediction-citation_network


## File Description

1. **cora_EDA.ipynb**
   - Exploratory Data Analysis (EDA) of the CORA citation network dataset.
   - Includes dataset statistics, graph properties, degree distribution, and visualizations.

2. **heuristics.ipynb**
   - Implementation of graph-theoretic link prediction methods.
   - Includes heuristic approaches such as Common Neighbors, Jaccard Coefficient, Adamic-Adar, Preferential Attachment, etc.

3. **gae_code.ipynb** *(Work in Progress)*
   - Implementation of a Graph Autoencoder (GAE) for link prediction on the CORA dataset.
   - Includes graph embedding generation and evaluation using metrics such as AUC and Average Precision.
